# EcoCollect

  A web application for managing waste collection efficiently. Admins, agents, and citizens can track, manage, and report waste collection tasks.

## Prerequisites
  - Node.js (>=18)
  - npm (comes with Node.js)

## Installation
    
  1. Clone the repository:
      
      git clone https://github.com/waste-collector/EcoCollect-Waste-Management-System.git  
      cd EcoCollect‑Waste‑Management‑System  
      
  2. Install dependencies:
      
      # Frontend root
      cd frontend
      npm install
      
      # Admin frontend
      cd frontend/admin
      npm install
      
      # Agent frontend
      cd ../agent
      npm install
      
      # Citizen frontend
      cd ../citizen
      npm install
      
## Running The Project
      
    # Admin frontend
    cd frontend
    npm run dev:admin
    
    # Agent frontend
    npm run dev:agent
    
    # Citizen frontend
    npm run dev:citizen
